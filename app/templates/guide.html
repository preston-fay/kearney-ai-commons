What's the Difference?

Prompts

Static text templates you copy and paste into any AI chat. One-time use, works everywhere.

  • Copy once, paste into AI
  • Works with any LLM
  • Great for quick tasks

Skills

Interactive, multi-step workflows. The AI asks questions and builds something custom for you.

  • Interview-style interaction
  • Generates custom outputs
  • Reusable workflows

Using Skills

Any LLM

Method 1: Copy & Paste (Works Everywhere)

  1. Find a skill and click "View Details"
  2. Select the "Any LLM" tab
  3. Copy the entire prompt
  4. Paste into your preferred AI (ChatGPT, Claude.ai, Gemini, etc.)
  5. Answer the interview questions the AI asks you
  6. Get your customized output!
Claude Code

Method 2: Install as Slash Command (Power Users)

  1. Find a skill and click "View Details"
  2. Select the "Claude Code" tab
  3. Copy the SKILL.md content
  4. Create the folder: ~/.claude/skills/[skill-name]/
  5. Save as SKILL.md in that folder
  6. In Claude Code, type /[skill-name] to run it!
Terminal
$ mkdir -p ~/.claude/skills/setup-workflow $ # Paste the SKILL.md content into this file: $ nano ~/.claude/skills/setup-workflow/SKILL.md $ # Now in Claude Code, just type: $ /setup-workflow
